# desci-connect

## Project Overview

This project is a scientific collaboration platform called DeSci Connect. It aims to connect researchers, facilitate the sharing of research papers, and provide a reputation system to incentivize quality contributions. The platform allows researchers to connect their wallets, submit research papers, interact with other researchers' posts, and mint their papers as NFTs.

## Installation Instructions

To set up the project locally, follow these steps:

1.  Clone the repository to your local machine.
2.  Navigate to the project directory.
3.  Run `npm install` to install the necessary dependencies.

## Usage Guide

### Running the Project

To run the project locally in development mode, use the following command:

```bash
npm run dev
```

This will start a local development server with hot-reloading.

To run the project in production mode, use the following command:

```bash
npm run start
```

This will start a local server using Vite's preview mode.

### Building the Project

To build the project for production, use the following command:

```bash
npm run build
```

This will create a `dist` directory containing the optimized production build.

### Running Tests

To run the test suite, use the following command:

```bash
npm run test
```

### Deployment

The project can be deployed to any static web hosting service. Simply upload the contents of the `dist` directory to your hosting provider.
